# 🎨 Momentum DS - Component Library

Welcome to **Momentum DS** Storybook! This is an interactive documentation for all React Native components designed for academy and fitness digital products.

## 📚 What is Storybook?

Storybook is a tool for building and testing UI components in isolation. You can:

- 🎯 **View all components** in one place
- 🎮 **Interact** with them using controls
- 📖 **Read documentation** for each component
- 💻 **Copy code** snippets and use them in your apps
- 🎨 **Test different states** (disabled, loading, etc)

## 🚀 Quick Start

### Using Momentum DS in Your App

```bash
npm install @andersonlucasp/rn @andersonlucasp/tokens
```

### Example Code

```tsx
import { DsButton, DsCard, DsBadge } from '@andersonlucasp/rn';
import { colors, spacing } from '@andersonlucasp/tokens';

export function MyApp() {
  return (
    <DsCard variant="elevated" padding="md">
      <DsText weight="semibold">Yoga Class</DsText>
      <DsBadge variant="success">Available (5)</DsBadge>
      <DsButton variant="primary">Reserve Class</DsButton>
    </DsCard>
  );
}
```

1. **Browse Stories** - Click on components in the left sidebar
2. **Interact** - Use the Controls tab to change props
3. **Copy Code** - Use the Show Code button to copy snippets
4. **Read Docs** - Each component has full documentation

## 🔧 Controls

Each story has interactive **Controls** where you can:

- Change the `variant` (primary, secondary, danger, etc)
- Adjust the `size` (sm, md, lg)
- Toggle `disabled`, `loading`, `required` states
- Change text and placeholders

Click a control value and watch the component update in real-time!

## 🎨 Design Tokens

All components use design tokens for colors, spacing, and typography:

```tsx
import { colors, spacing, fontSize } from '@andersonlucasp/tokens';

const styles = StyleSheet.create({
  container: {
    backgroundColor: colors.background.primary,
    padding: spacing.lg,
  },
  text: {
    color: colors.text.primary,
    fontSize: fontSize.base,
  },
});
```
