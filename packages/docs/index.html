🎨 Momentum DS - Component Library

Welcome to Momentum DS Storybook!

This is an interactive component library for React Native apps in the academy and fitness industry.

🚀 Quick Start

Install the package:

npm install @andersonlucasp/rn @andersonlucasp/tokens

Use a component:

import { DsButton, DsCard } from '@andersonlucasp/rn';\n\n<DsButton variant="primary">Click me</DsButton>

📚 Navigation

Click on any component in the left sidebar to see it in action.

Each story includes:

  • 🎮 Interactive canvas
  • 🎛️ Controls to change props
  • 💻 Code preview
  • 📖 Full documentation
